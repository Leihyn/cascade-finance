## âš¡ Installation & Usage

### Step 1: Choose a Theme
Look at previews above or read full specs in `FRONTEND_DESIGN_VARIATIONS.md`

### Step 2: Switch Theme

**Option A: Using Scripts (Recommended)**
```bash
# Windows
scripts\switch-theme.bat terminal

# Mac/Linux
./scripts/switch-theme.sh cyberpunk
```

**Option B: Manual Copy**
```bash
# Backup current
cp frontend/app/globals.css frontend/app/globals.css.backup

# Copy new theme
cp frontend/styles/themes/defi-cyberpunk.css frontend/app/globals.css

# Add Tailwind directives at top
# (see THEME_SWITCHER_GUIDE.md)
```

### Step 3: Install Fonts (if needed)

Some themes need additional fonts:

```bash
cd frontend

# For Cyberpunk
npm install @fontsource/orbitron @fontsource/jetbrains-mono

# For Terminal
npm install @fontsource/fira-code @fontsource/jetbrains-mono

# For Maximalist
npm install @fontsource/poppins @fontsource/dm-sans

# For Neumorphic
npm install @fontsource/nunito @fontsource/work-sans
```

Or use Google Fonts CDN (already included in theme CSS).

### Step 4: Restart Dev Server
```bash
cd frontend
npm run dev
```

Visit http://localhost:3000 to see your new theme!

## ğŸ”§ Customization

### Adjust Colors

Edit theme file in `frontend/styles/themes/`:

```css
:root {
  --bg-primary: #YOUR_COLOR;    /* Change background */
  --accent-primary: #YOUR_COLOR; /* Change accent */
  /* ... */
}
```

### Mix Themes

Combine elements from multiple themes:

```css
/* In globals.css */
@import "../styles/themes/classic-banking.css";

/* Add Terminal-style data displays */
.data-display {
  font-family: 'JetBrains Mono', monospace;
  color: #00FF41;
  background: #000;
}
```

### Create Custom Theme

1. Copy existing theme as starting point
2. Modify CSS variables
3. Adjust component styles
4. Test thoroughly
5. Share with community!

## ğŸ†˜ Troubleshooting

**Theme not applying?**
- Clear browser cache (Ctrl+Shift+R)
- Restart dev server
- Check console for CSS errors

**Fonts not loading?**
- Install fontsource packages: `npm install @fontsource/[font-name]`
- Or use Google Fonts CDN (already in CSS files)

**Animations laggy?**
- Reduce `blur()` values in theme CSS
- Disable animations for `prefers-reduced-motion`
- Test on different devices

**Colors look wrong?**
- Verify CSS variables in :root
- Check for browser extensions interfering
- Test in incognito mode
